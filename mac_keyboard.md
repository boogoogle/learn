使用mac键盘的一些习惯设置

1. 设置CapsLock为cmd键盘,
   1. 因为用大写的时候少,按住shift,再按对应的字母就可以实现输入大写
2. cmd + n 新建chrome窗口
3. cmd + t 在当前窗口新建标签页
4. cmd + w 关闭当前tab,
   1. 如果只有一个tab页,则关闭当前应用程序(相当于cmd + q)
   2. vscode, iterm 也同样适用
5. cmd + 方向上, 浏览器滚动到顶部; 
   1. 反之滚动到页尾
   2. vscode 同样适用
6. cmd + tab 切换打开的应用程序到前台(可以使用方向左键向左切换)
7. cmd + 空格, 快速打开某个应用
